# Rust Manual for Rust Malaysia

Simple installation guide with rust basics 


Make sure typst is installed

```
cargo install --locked typst-cli
```


Supported Operation:

To make a full pdf:
```
make all
```


To make a CheatSheet pdf only:
```
make cheatsheet
```

to clean:
```
make clean
```

realtime compiling:
```
make watch
```
